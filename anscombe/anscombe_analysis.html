Análisis del Cuarteto de Anscombe¶

Introducción¶

El Cuarteto de Anscombe es un conjunto de cuatro conjuntos de datos que tienen las mismas estadsticas descriptivas, pero diferentes distribuciones y visualizaciones. Este anlisis se centrar en explorar estos conjuntos de datos utilizando grficos de dispersin y box-plots para demostrar cmo las estadsticas pueden ser engaosas si se presentan sin el contexto visual adecuado.

En este notebook, se cargar un dataset que contiene los cuatro conjuntos de datos de Anscombe, se realizar un anlisis estadstico y se visualizarn los datos a travs de grficos.

Gráfico de Dispersión por Grupos del Cuarteto de Anscombe¶

In [23]:
# Creamos una cuadrcula de grficos (FacetGrid) para los datos
g = sns.FacetGrid(anscombe, col='group', hue='group', palette='deep', col_wrap=2, height=5, aspect=1.2)
# Grfico de dispersin para cada grupo de datos
g.map(sns.scatterplot, 'x', 'y', s=100)  # s es el tamao de los puntos

g.set_titles(col_template='Grupo {col_name}')

g.add_legend()  # Agregamos una leyenda para identificar los grupos

plt.tight_layout()  # Ajustamos el espacio entre los grficos
plt.show()

# Creamos una figura para el boxplot
plt.figure(figsize=(12, 6))  # Tamao de la figura
# Creamos un boxplot para mostrar la distribucin de 'y' segn el grupo
sns.boxplot(x='group', y='y', data=anscombe, hue='group', palette='deep', legend=False)
plt.title('Boxplot')  # titulo al boxplot
plt.tight_layout()  # Ajustamos el espacio en el grfico
